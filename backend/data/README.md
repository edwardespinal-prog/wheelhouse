# Data Directory

This directory stores your portfolio data as JSON files. All files are created automatically on first run.

**Do not commit data files to git** — they are excluded via `.gitignore`.

## Files created automatically
- `equities.json` — Stock positions
- `options.json` — Options positions
- `crypto.json` — Cryptocurrency positions
- `tax_ledger.json` — Realized gains/losses
- `cc_income.json` — Covered call premium tracking
- `cards.json` — Collectibles inventory (Uncle Eddi3's)
- `watchlist.json` — Watchlist tickers
- `cash_flow.json` — Deposit/withdrawal history
- `cash_balance.json` — Current cash balance
- `summary.json` — Portfolio summary cache
- `snapshots.json` — Daily portfolio snapshots
- `wheelhouse_config.json` — Tab and feature configuration
